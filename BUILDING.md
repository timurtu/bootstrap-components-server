## Run the demo app locally
- `git clone https://github.com/timurtu/bootstrap-components`
- `cd bootstrap-components/demo-app`
- `npm install`
- uncomment the bundle script reference in __index.html__

then either 
- `npm run prod`

or

- `cd ../bootstrap-components-server`
- `node index.js`
